### Приложение мини-канбан
Можно перетаскивать заданчи между тремя колонками

### Запуск
Импортируйте репозиторийку
Установите react и yarn
Используйте команду `yarn start` ждя запуска.
Приложение будет работаьть на 3000 порту. (http://localhost:3000)
